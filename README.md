# 체스 게임

## 환경 설정

python 3.9.13을 사용합니다.

```shell
pyenv install 3.9.13
pyenv local 3.9.13
python -V
```

poetry를 사용해 가상환경을 관리합니다.

```shell
poetry install
```

pre-commit으로 commit 전

```shell
pre-commit install
```
